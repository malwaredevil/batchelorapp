# Batchelor App

A personal household app combining pottery and quilting collection management, travel planning, and an AI assistant — all under one login.

## What it does

- **Pottery** — catalogue and AI-search a pottery collection (photos, categories, semantic search)
- **Quilting** — catalogue fabrics, patterns, and finished quilts; plan quilt layouts; track a shopping list
- **Travels** — plan and track trips, reminders, packing lists, and travel documents; integrates with Google Calendar and Gmail
- **Ornaments** — catalogue a Christmas ornament collection with UPC barcode lookup and AI analysis
- **Elaine** — an AI assistant with full access to all household data, available via the web app, SMS, and email

## Stack

Node.js 24 · TypeScript 5.9 · Express 5 · PostgreSQL (Supabase) · React 19 · Vite 7 · Drizzle ORM · OpenRouter · pnpm workspaces

## Status

This is a private household application. It is not open for public contributions.
